# **App Name**: Nmouk

## Core Features:

- Intelligent Onboarding Chat: Engage users through an interactive chat interface to collect crucial business information via 10 detailed questions, forming the foundation for AI analysis.
- AI Reality Check Tool: Implement a 'reality check' tool using AI to assess user input for aspirational statements and prompt for actual, current business status, ensuring realistic data for analysis.
- Brand Identity Generation & Selection: Leverage Google Gemini API to analyze user responses, proposing three distinct brand identity options complete with Archetype, Plot, global references, narrative, and clear justification for each, recommending the optimal choice.
- User-Driven Brand Choice: Provide a user interface for reviewing the three AI-generated brand identity options and selecting the preferred one, with an option to define a custom identity.
- Comprehensive Document Synthesis: Generate three essential marketing documents: Brand Identity, Go-To-Market Strategy, and Marketing Charter, populated with context-aware content derived from user inputs and advanced AI analysis.
- Interactive Charter Revision: Allow users to review the AI-generated Marketing Charter, provide specific feedback, and utilize the Gemini API to intelligently revise and regenerate the document based on their comments.
- Persistent Session Management: Implement robust session management using localStorage for user session IDs and Supabase for all accumulated user data and progress, enabling seamless continuation across different visits.

## Style Guidelines:

- The application employs a dark color scheme for a professional and sophisticated user experience, conducive to focused analytical work.
- Primary color: (#194b35) reflecting intelligence and professional insight..
- Background color: (#fbf8f3) to enhance readability and contrast in dark mode..
- Accent color: (#4f4f4f) for interactive elements, highlights, and calls to action, providing clear visibility against the light background..
- Headline font: 'Space Grotesk' (sans-serif) to convey a modern, tech-driven, and analytical feel for titles and prominent text.
- Body font: 'Tajawal' (sans-serif) for high legibility and a neutral, professional appearance, ideal for extensive text like chat responses and generated documents.
- The application supports both RTL (Arabic) and LTR (English) text direction, requiring fonts compatible with both scripts for comprehensive language support.
- Utilize clean, minimalist line-art or filled outline icons for intuitive navigation and interaction, consistent with a professional, data-centric interface.
- Employ a responsive layout design that intelligently adapts for both LTR and RTL language orientations, ensuring optimal readability and usability across all content areas.
- Incorporates subtle, functional animations for transitions, content loading, and feedback on user actions, maintaining a polished and non-distracting user experience appropriate for a professional tool.
